  NDI Preview System:

  - ndi/preview.go — Previewer interface with Start(ip), Stop(), Frame(),

  - ndi/stub.go — Default stub (no NDI SDK needed). Preview shows "No Signal"
  - ndi/receiver.go — Full CGo NDI receiver (build with -tags ndi). Uses libndi
  to discover NDI sources by camera IP, receives low-bandwidth preview frames in
   RGBA, and encodes them as JPEG
  - GET /api/preview — WebSocket endpoint that streams binary JPEG frames at
  ~10fps
  - Preview box in the UI above the D-pad — 16:9 black box with live feed or "No
   Signal" text. WebSocket auto-reconnects every 3 seconds if disconnected
  - NDI receiver starts automatically when a camera connects (by IP) and stops
  on disconnect

  To use with NDI:
  go build -tags ndi .
  Requires libndi installed (from https://ndi.video/tools/ — free download).
  Without the tag, everything compiles and runs normally; preview just shows "No
   Signal."
